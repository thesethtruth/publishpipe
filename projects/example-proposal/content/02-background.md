## Background

Publishpipe converts markdown files into print-ready PDFs using:

- **Nunjucks** templates for layout
- **CSS paged media** for pagination and print styling
- **pagedjs-cli** for PDF rendering

### Key Features

1. Multi-chapter documents from separate markdown files
2. Title page with configurable metadata
3. Light and dark theme support
4. Configurable page size and margins
5. Hot reload during development

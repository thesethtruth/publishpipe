# Overview
A productive and highly customizable way of getting PDF documents out of the door that are completely in sync with the rest of your digital presence.

## Goal
- Write content in markdown
- Make templates in HTML/CSS/Nunjucks
- Convert markdown to html using Pandoc?
- Bake content to nice PDFs using pagedjs-cli

## Stack
- Tailwind CSS
- Bun

## Level of detail
- Swapable templates
- Easy to change stylesheets
